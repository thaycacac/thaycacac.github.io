* Responsive sẽ ẩn đi khi vượt quá số dùng
```css
  display: -webkit-box;
  -webkit-line-clamp: 1;
  -webkit-box-orient: vertical;
  overflow: hidden;
  text-overflow: ellipsis;
```

* Flex để center
```css
  display: flex;
  align-items: center;
  justify-content: center;
```
